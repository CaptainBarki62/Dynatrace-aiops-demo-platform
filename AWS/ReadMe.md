# Dynatrace AIOps Self-Healing Lab

Enterprise observability and automation lab built on AWS EKS and Azure AKS, showcasing Dynatrace AIOps, Kubernetes monitoring, OpenTelemetry, workflow automation and self-healing operations.

## Objectives

- Dynatrace Operator Deployment
- Kubernetes Monitoring
- EasyTrade Deployment
- Davis AI Problem Detection
- Synthetic Monitoring
- SLO Management
- Workflow Automation
- Auto Remediation
- ServiceNow Integration
- OpenTelemetry Collector Evaluation

## Platforms

- AWS EKS
- Azure AKS
- Dynatrace SaaS
- Kubernetes
- OpenTelemetry
